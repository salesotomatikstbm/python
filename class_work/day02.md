## Apple Brochure in Python

### Python Program

```python
fruit_name = "apple"
fruit_color = 'red'
quantity_kg = 6
price_rupees = 120.5
healthy = True
````

---

### Data Types

| Example values                            | Data type |
| ----------------------------------------- | --------- |
| `"apple"`, `"red"`, `"my name is ramesh"` | `str`     |
| `12`, `18000`, `-250`                     | `int`     |
| `120.5`, `3.14`, `-0.5`                   | `float`   |
| `True`, `False`                           | `bool`    |

---

### Checking Data Types

```python
print(type(fruit_name))

```
